## 1. 문서 개요

### 1.1 목적
본 문서는 요구사항 정의서에서 정의한 기능/비기능/LLM 연동 요구사항 및 추적성 매트릭스를 기준으로, 테스트 범위·시나리오·기대 결과를 사전 정의한 v0.1 계획과, 이를 실제 환경에서 실행한 결과를 하나의 문서로 통합하여 요구사항-화면설계-개발-테스트 간 추적성을 확보하는 것을 목적으로 한다.

### 1.2 테스트 범위
사용자 화면 6종(홈, QA, 권리찾기, 증거자료 관리, 계산기, 최신 뉴스), 관리자 콘솔 5종(대시보드, 사용자 관리, 피드백 관리, 프롬프트 관리, 성능 모니터링), 마이페이지, LLM/외부 API 연동(OpenAI GPT, OpenAI Embedding, 네이버 뉴스 API), 배포 환경(Docker/AWS)을 포함한다.

### 1.3 테스트 환경

| 항목 | 내용 |
|---|---|
| 백엔드 | Python 3.10, Django (MVT), LangGraph, ChromaDB |
| 프론트엔드 | Django Template (데스크톱 1920px / 모바일 360~768px) |
| 외부 연동 | OpenAI GPT / Embedding API, 네이버 뉴스 API |
| 배포/인프라 | Docker, docker-compose(app+DB+Nginx), AWS(ECS/EC2, RDS/벡터DB, S3, ALB), CI/CD |
| 실행 대상 브랜치/버전 | 로컬 개발 브랜치 (배포 검증 시 `main` → `origin/deploy` 실제 push 포함) |
| 실제 실행일 | 2026-07-08 |

### 1.4 테스트 전략
- 요구사항 정의서 6장 추적성 매트릭스의 TC ID 체계를 그대로 준용하여 FR/NFR/LLM 요구사항과 1:1 매핑되도록 테스트케이스를 구성한다.
- 인증, 폼 검증, CRUD 등 핵심 기능은 정상 케이스와 예외 케이스(Edge case)를 구분하여 작성한다.
- LLM API 연동은 비동기 통신 특성을 고려하여 정상 호출뿐 아니라 타임아웃, 재시도, 상태코드별 예외 처리까지 검증한다.
- 배포 환경은 서버 재기동 및 컨테이너 재현성 관점에서 데이터 영속성과 구동 일관성을 검증한다.
- 발견된 이슈는 증상-원인-수정-재검증의 흐름으로 추적 가능하게 기록한다.
- **(실행 단계 추가)** 실인프라(AWS/실도메인)가 필요한 항목은 자체서명 인증서 및 별도 테스트용 compose 파일로 로컬에서 재현하여 실측 검증하고, 실제 실인프라 검증(GitHub Actions 배포)도 사용자 확인 후 1건 수행하였다.

### 6.4 정정된 항목 (기존 "미구현" 판단을 뒤집음)
- **권리찾기(FR-030~032)·증거자료 체크리스트(FR-050~051)**: 별도 전용 화면은 없지만, `/app/?page=home` AI 상담 채팅을 통해 실제로 기관 안내·절차·증거자료 체크리스트가 모두 노출되고 이력도 저장됨을 실측으로 확인. "미구현"이 아니라 "전용 화면 없이 통합 채팅으로 커버"가 정확한 상태.

### 6.5 이번에 별도로 해결된 항목
- **S3 / HTTPS**: `origin/deploy` 병합으로 실제 도메인 HTTPS(Let's Encrypt)와 배포 시 S3 vector DB 동기화가 설정에 반영됨.

- **TC-DEPLOY-03**: 사용자 확인 후 실제 `main`→`origin/deploy` push로 GitHub Actions 배포 워크플로우를 실제 트리거하여 성공 확인. 프로덕션 서버에서 HTTPS(443, 실제 Let's Encrypt 인증서)까지 정상 동작함을 직접 확인 완료 — 실인프라 관련 항목 중 미검증으로 남은 것은 없음.

### 6.6 새로 발견된 배포 파이프라인 개선 여지
- **TC-DEPLOY-03 헬스체크 커버리지**: `.github/workflows/deploy-ec2.yml`의 배포 후 헬스체크가 `http://localhost/`(80번 포트, HTTP)만 확인하고 443(HTTPS) 자체는 검증하지 않음. HTTPS 설정이 깨진 채로 배포되어도 헬스체크는 통과할 수 있음 (이번엔 실제로 443도 정상이었으나, 스크립트 자체의 커버리지 공백은 남아있음).
